# 📝 Project Overview

The **Invoice Management System** is a full-stack web application designed to manage invoices, products, and financial records for businesses.

The system allows users to:

* Create and manage invoices
* Track invoice payment status
* Manage product sections
* Upload invoice attachments
* Generate financial reports
* Export reports to Excel

This project demonstrates real-world implementation of **Laravel MVC architecture, authentication systems, database management, and business logic**.

---

# ✨ Key Features

### 📄 Invoice Management

* Create, update, delete invoices (**CRUD operations**)
* View detailed invoice information
* Print invoice details

### 💰 Payment Status Tracking

* Paid invoices
* Unpaid invoices
* Partially paid invoices

### 📦 Products & Sections

* Manage product sections
* Assign products to sections
* Dynamic product loading based on section

### 📎 Invoice Attachments

* Upload files related to invoices
* View attachments
* Download files
* Delete attachments

### 📊 Reports System

* Invoices report
* Customers report
* Search invoices by filters

### 📤 Data Export

* Export invoices to **Excel**
* Export reports to **Excel**

### 🗂 Archive System

* Archive invoices
* Restore archived invoices
* Permanently delete archived invoices

### 🔐 Security Features

* **Authentication system**
* **Authorization using roles**
* Email verification
